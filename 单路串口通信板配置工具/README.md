# 传感器配置工具

搭配单路串口通信板使用
可以开关传感器数据传输和使能相关处理算法

## 使用方法

1.选择对应串口，点击连接

2.点击START，启动传感器数据传输(部分传感器可能已经开启，此时此按钮状态可能显示异常，点击即可恢复)

3.点击GET获取到传感器的状态位，若某状态已经开启则下方按钮会高亮。

4.选择需要打开的处理算法(高亮为打开)和映射放缩的大小,点击SET即可设置。

5.数据漂移可以点击CALIB重新进行标定。



## 状态说明

标定：输出传感器与标定值的变化量

死区过滤：过滤掉变化量小于死区值的数据

滑动窗口：平滑传感器数据

预测补偿：使用参考传感器进行磁场屏蔽

映射：将标定后的变化量映射到力值范围，乘以X，Y，Z框中的对应值

平滑过渡：将稳定后的漂移数据平滑到0

## 工具列表

### 单路串口通信板配置工具
传感器配置工具，搭配单路串口通信板使用。

**功能特性：**
- 开关传感器数据传输
- 使能相关处理算法

**使用方法：**
运行 `单路串口通信板配置工具/传感器配置工具.exe`

### 通用传感器数据绘图
可以绘制多路三轴传感器数据的可视化工具。

**数据格式：**
```
sensor{n}:{x},{y},{z}
```

**示例：**
```
sensor0:199,299,399
```

**使用方法：**
运行 `通用传感器数据绘图/数据绘图.exe`

**注意事项：**
- `_internal` 文件夹必须与可执行文件在同一目录下
